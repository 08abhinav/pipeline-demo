# pipeline-demo

A hands-on DevOps project demonstrating how to design, implement, and document a complete CI/CD pipeline for a web application, following real production-style workflows.

---

## Project Overview

This project showcases the complete setup of a CI/CD pipeline from scratch. Over 7 days, the pipeline was designed to automate the **build, test, and deployment** of a web application using real-world DevOps tools and best practices.

Key goals:

- Automate application build and deployment
- Implement continuous integration and delivery
- Use containerization for consistent environments
- Document each step for portfolio and learning purposes

---

## Tech Stack

- **Version Control:** Git & GitHub  
- **CI/CD:** Jenkins
- **Web Application:** Java
